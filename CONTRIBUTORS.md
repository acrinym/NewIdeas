## How to Be Listed

### For Code Contributors
- Submit a pull request that gets merged
- You'll be automatically added to this file
- Include your GitHub username and brief description

### For Theme/Plugin Creators
- Publish a theme or plugin to the marketplace
- Submit a PR adding yourself to the "Theme & Plugin Creators" section
- Include links to your creations

### For Documentation Contributors
- Improve docs, write tutorials, fix typos
- Submit a PR with your changes
- Add yourself to "Documentation Contributors"

### For Community Contributors
- Help users in issues/forums
- Report bugs with detailed reproduction steps
- Test new features and provide feedback
- Active for 3+ months? Add yourself!
